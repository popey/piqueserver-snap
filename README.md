## Piqueserver

An Ace of Spades 0.75 server based on PySnip.

This snap is a community-maintained unofficial build of [piqueserver](https://github.com/piqueserver/piqueserver).

The current refresh work tracks upstream `piqueserver` `v1.3.0`, moves the snap to `core24`, and adds GitHub Actions automation that builds, reviews, installs, and smoke-tests the service on branch pushes.

[![Get it from the Snap Store](https://snapcraft.io/static/images/badges/en/snap-store-black.svg)](https://snapcraft.io/piqueserver)

## Install

Install on any [supported Linux](https://snapcraft.io/docs/installing-snapd) distribution:

```
snap install piqueserver
```
